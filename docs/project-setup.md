## Project Setup

Run these commands to create and start the Expo project:

```bash
npx create-expo-app@latest one-dex   # create a new Expo app named "one-dex"
cd .\one-dex\                        # move into the project folder
npx expo start a                     # start Expo and run the app on Android
npx expo install expo-apple-authentication expo-application expo-crypto expo-linking expo-secure-store expo-web-browser react-native-passkeys react-native-webview @privy-io/expo-native-extensions @privy-io/expo # install dependencies (Expo-managed versions)
```
